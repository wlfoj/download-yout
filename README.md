# Informações
Se não precisa do console, use o arquivo ´direct.py´ basta colocar a url em `URL`

## Requerimentos minimos do ambiente
```$ python3 -m venv env.```
```$ pip install -r /path/to/requirements.txt```
## Ativando o ambiente
### Windows
```$ env\Scripts\activate.bat```
### Linux ou Mac
```$ source env/bin/activate```
